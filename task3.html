Задание 3. Flexbox.

В этом задании нужно было создать элемент Flexbox из четырех фреймов которые меняются при наведении/нажатии по ним кнопкой мыши.

Flexbox — анимация из 4 фреймов

Последовательность для смены фреймов и демонстрации:

1. Наведите курсор мышки

2. Совершите клик мышкой по фрейму.

3. Совершите клик мышкой по фрейму с удержанием кнопки.

4. Отпустите кнопку мышки.

2. Совершите клик мышкой по фрейму для перезапуска.
