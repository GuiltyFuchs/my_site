Задание 1

Сайт должен демонстрировать адаптивность, модульную структуру и чистый дизайн.

Причины выбора стиля

Минималистичный и модульный стиль выбран для фокуса на контенте, легкости масштабирования и адаптивности. Приоритет на читабельность (контрастные цвета, шрифты Inter), доступность и предсказуемую сетку. Цветовая палитра: акценты для ссылок, градиенты в логотипе для визуального интереса без перегрузки.

Как реализовано

Структура страницы разделена на семантические блоки: header, main (с секциями), footer. Для вёрстки использованы CSS Grid и Flexbox: Grid отвечает за глобальную раскладку, Flexbox — за внутри-блоковые выравнивания. Для адаптивности применены медиазапросы и max-width для изображений. Навигация превращается в «гамбургер» на узких экранах (JS переключает aria-expanded и класс).

Стили, техники и формы

  • CSS Grid — для главной раскладки блоков и изображений.
  • Flexbox — для горизонтальных списков и кнопок.
  • Медиазапросы на 1024px, 800px и 480px.
  • SS-логотип с градиентами, тенями и псевдоэлементами (размер 48px, адаптивный).
  • Адаптивные изображения с max-width:100%, object-fit.

Длинный текст и иллюстрации (более 300 слов + 5 изображений)

В процессе создания этого сайта я старался делать его простым и понятным, чтобы всё было аккуратно организовано. Я выбрал минималистичный стиль: убрал лишние элементы и оставил только важное. Так контент легче воспринимается, а страницы выглядят чисто и аккуратно.

Структура сайта делится на блоки: шапка с логотипом и меню, основное содержимое и подвал (footer). На главной странице описана цель сайта и планы на будущее. Страница с заданиями показывает, как выполнены учебные задачи: есть список заданий, отдельные страницы для каждого, и кнопки для перехода к демонстрациям.

Для верстки я использовал CSS Grid и Flexbox: Grid отвечает за общую раскладку блоков, а Flexbox помогает выравнивать элементы внутри секций. Сайт адаптивный — при уменьшении ширины экрана блоки и изображения перестраиваются так, чтобы всё оставалось читабельным.

Каждый элемент продуман: заголовки, абзацы и картинки занимают своё место, цвета и отступы согласованы. Все изображения имеют alt-текст для доступности, ссылки и кнопки работают с клавиатуры, а визуальные эффекты не отвлекают от основного контента.

Я также подготовил собственный логотип на CSS, использовал единый цвет и шрифты по всему сайту, чтобы дизайн был целостным. На странице с длинным текстом размещено не менее 300 слов и несколько изображений, чтобы проверить поведение блоков при разных размерах экрана.

В будущем сайт можно будет подключить к серверной логике, но уже сейчас структура понятная и модульная: стили, компоненты и шаблоны легко повторно использовать и масштабировать.
