Задание 2. CSS-рисунок

В этом задании требовалось создать изображение размером 200×200 пикселей, полностью средствами CSS, без использования растровых изображений.

Описание задания

Цель задания — продемонстрировать возможности CSS при создании графических элементов с использованием блочной модели, градиентов, теней и псевдоэлементов.

Как реализовано

Изображение собрано из нескольких элементов div. Для оформления применялись фоновые градиенты, скругления углов, тени и псевдоэлементы ::before и ::after. Все размеры заданы относительно контейнера 200×200 пикселей.
