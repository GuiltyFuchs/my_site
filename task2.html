Задание 2. Рисунок CSS.

В этом задании нужно было создать изображение размером 200х200 пикселей, полностью используя CSS без применения картинок. За основу был взят предложенный логотип-пример.

Для выполнения задания был выбран способ сборки изображения из простых геометрических блоков. Рисунок создавался из нескольких элементов div с использованием фоновых заливок, границ и псевдоэлементов ::before и ::after. Цвета задавались при помощи градиентов, а для создания эффекта объёма применялись тени и скругления углов. Все размеры элементов подбирались относительно основного контейнера размером 200×200 пикселей, чтобы рисунок соответствовал требованиям задания. Такой подход позволил получить изображение полностью средствами CSS без использования растровых картинок.
