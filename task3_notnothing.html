Задание 3. Flexbox.

В этом задании нужно было создать элемент Flexbox из четырех фреймов которые меняются при наведении/нажатии по ним кнопкой мыши.

Flexbox — анимация из 4 фреймов

Для смены первого фрейма наведите на него мышкой.
Последующие изменения фреймов происходят при нажатию мышкой по фрейму.
